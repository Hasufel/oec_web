Habitable zone

The following plot shows the approximate location of the planets in this system with respect to the habitable zone (green) and the size of the star (red). This is only an estimate, using the star's spectral type and mass. Note that if no green band is shown in the plot, then the planet's orbit is far outside the habitable zone. The equations of Selsis, Kasting et al are used to draw the inner and outer boundaries.

Corrections

If you spot an error or if you can contribute additional data to this entry, please send an e-mail to exoplanet@hanno-rein.de. Please include the corrected xml file and a reference to where the new data is coming from, ideally a scientific paper. If you are fluent with git and github, you can also create a pull request or open an issue on the Open Exoplanet Catalogue repository.
